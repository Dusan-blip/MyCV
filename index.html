3D Rendering pipeline implemented in C. Using the SFG library’s functionality to draw
triangles on the screen, this projects creats a pseudo 3D game engine with a movable camera

a Z Buffer, a simple lambert shader with colors truncated to library acceptible values.
https://github.com/Dusan-blip/Kosilica

Fantasy clover shader. Inspired by a scene from a game called One Shot, This material only
has color when reflecting light. The shader is made in Unreal Engine and implements custom
reflection calculations as well as additional stylized lighting with the use of the custiom node and
HLSL programming. https://dusan-blip.github.io/clover/

BlackJack in Assembly x86. With the use of the Irvine library, this project recreates the game
of Blackjack in the windows terminal using ASCII art. It supports the creation of custom ASCII
sprites and animations, Procedural generation of cards and has an AI which can play against up
to 3 players. https://github.com/Dusan-blip/asm-cmd-blackjack
•
Strategy game system and animation handling in Unreal Engine. In this Project I created the
logic behind a turnbased strategy game as well as the animations and their handling by the
engine. Animations use animation montages, notify events, blend spaces, and custom animation
blueprints, whereas the game code is modular and supports easy creation of additional units and
abilities, status effects, day/night cycle events and so on. I also made the cutscenes using UE5’s
level sequences. https://dusan-blip.github.io/KingsSquable/
